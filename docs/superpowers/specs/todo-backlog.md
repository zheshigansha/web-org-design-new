## 暂不调整的功能

**前置条件：** 接入第三方平台 API 或 爬虫服务
**优先级：** P2

### 1.5 提醒通知
**原因：** 需要消息推送基础设施（邮件/企微/钉钉）
**前置条件：** 消息推送服务接入
**优先级：** P3
**包含功能：**
- 每日报告生成通知
- 热度异动提醒
- 选题池待处理提醒
- 定时推送

### 1.6 会员系统
**原因：** 需要用户认证/权限/订阅管理等完整系统
**前置条件：** 多用户架构设计
**优先级：** P3
**包含功能：**
- 多用户/团队协作
- 角色权限（管理员/编辑/查看）
- 消费积分（AI报告次数限制）
- 订阅套餐
